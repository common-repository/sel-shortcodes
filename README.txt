This plugin created for official themes from Selthemes.com

== Description ==
This plugin generates following shortcodes on wp sites.
* Grid
* Button
* Callouts
* Accordion
* Tabs


== Frequently Asked Questions ==

= Installation Details =
1. Upload 'sel-shortcodes' to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress.

=Can I use this plugin with “non-selthemes” themes?=
Yes.
